reactgulp
===
gulp environment for simple services

## Usage

### Preparing
    npm run setup

### Begin to watching files
    npm start

### Minify for release
    npm run package

## Dependency services
- [Gulp](http://gulpjs.com/)
- [WebPack](http://webpack.github.io/)
- [Bower](http://bower.io/)
